A platexmkrc file for use with the pdflatexmk engine in TeXShop. Place the line `% !TEX program = pdflatexmk` at the top of your Source file. Add the platexmkrc file to the same folder as your Source file. finally, just typeset using Typeset->Typeset (Cmd-T) and all the processing will get done automatically. NOTE: the platexrc file assumes you are using anaconda2 python; change the single occurance of `anaconda2' to `anaconda3' in that file if you are actually usisng anacodna3 python.
